# daign-style-sheets

[![NPM package][npm]][npm-url]

Simple style sheet processor.

## Installation

```sh
npm install @daign/style-sheets --save
```

## Scripts

#### Build

    npm run build

#### Run lint analysis

    npm run lint

#### Run unit tests with code coverage

    npm run test
